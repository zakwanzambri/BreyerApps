# Campus Hub - Modern College Student Portal

Welcome to Campus Hub, a comprehensive web application designed to centralize all campus resources and services for college students in one beautiful, easy-to-use portal.

## 🎯 Purpose

Campus Hub solves the common problem of scattered information and resources that students face daily by providing:

- **Centralized Access**: All campus services and resources in one location
- **Personalized Dashboard**: Customized content based on student needs
- **Mobile-First Design**: Responsive interface that works on all devices
- **Real-Time Updates**: Live campus information and notifications
- **Intuitive Navigation**: Easy-to-use interface with powerful search

## ✨ Features

### 🏠 Dashboard
- Personalized welcome screen with quick stats
- At-a-glance overview of assignments, exams, and deadlines
- Weather and campus status information

### 📅 Academic Calendar
- Upcoming assignments and exams
- Class schedules and deadlines
- Important academic dates

### 📚 Course Materials
- Direct access to course syllabi

### 🎯 Campus Services
- Dining hall hours and menus

- Parking information

### 📢 News & Announcements
- Campus news and updates
- Important announcements

### 🔗 Quick Links
- Student portal access
- Campus email integration
- Learning Management System (LMS)
- Financial aid information

### 🔍 Global Search
- Search across all campus resources
- Smart suggestions and autocomplete
- Quick access to frequently searched items

## 🛠️ Technical Stack

- **HTML5**: Semantic markup for accessibility

- **Vanilla JavaScript**: Interactive functionality

## 🚀 Getting Started

1. **Setup**: Place the Campus Hub files in your web server directory
2. **Access**: Navigate to `index.html` in your web browser
3. **Customize**: Modify content to match your institution's needs

### File Structure
```
campus-hub/
├── index.html          # Main application file
├── css/
│   └── styles.css      # Complete styling
├── js/
│   └── main.js         # Interactive functionality
├── images/             # Image assets
└── README.md           # This file
```
